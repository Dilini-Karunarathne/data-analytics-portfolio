# data-analytics-portfolio
Power BI dashboards for data analytics portfolio


# Project Overview

This repository contains a collection of four end-to-end Power BI dashboards developed for business intelligence, data analytics, and performance reporting. These dashboards convert raw datasets into meaningful insights to support strategic and operational decision-making.

This project demonstrates strong practical skills in data analysis, visualization, KPI development, business reporting, and data modeling using Power BI.

Key skills applied in this project include data modeling using Star Schema design, DAX calculations for advanced metrics, and interactive dashboard development for business insights and decision support.

# Sales Dashboard (Business KPI Reporting)

This dashboard focuses on sales performance tracking and revenue analysis. It provides insights into total sales, profit, order volume, and growth trends.

It identifies top-performing products and categories and helps stakeholders understand overall business performance. The dashboard supports data-driven decision-making by highlighting key sales patterns and trends.

# HR Analytics Dashboard & HR OLAP Dashboard (Customer Segmentation & Behavior Analysis)

This dashboard analyzes customer behavior and purchasing patterns to support customer segmentation. It provides insights into customer trends and buying habits.

The analysis supports CRM strategies and marketing optimization by identifying valuable customer groups. It also enhances customer retention insights through behavioral analysis.

# Sales & Risk Analysis Dashboard (DAX-Based Risk Analysis)

This dashboard uses advanced DAX measures to classify business risk levels. It identifies loss-making orders and high-risk transactions.

Risk levels are categorized as Low, Medium, High, and Critical. This dashboard supports operational risk monitoring and helps improve business control and decision-making.


# Tools & Technologies

Power BI Desktop for data visualization and reporting

DAX (Data Analysis Expressions) for KPI calculations and advanced metrics

Power Query for data cleaning and transformation

Data modeling using Star Schema and relational structures

Excel and CSV datasets for data sources

Business Intelligence (BI) reporting techniques for analytical storytelling
